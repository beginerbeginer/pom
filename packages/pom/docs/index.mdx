# pom

**pom (PowerPoint Object Model)** is a library for declaratively describing PowerPoint presentations in TypeScript. It takes XML as input and generates editable PowerPoint files (.pptx).

[Try in Playground →](/playground)

![Playground](./images/playground.png)

## Features

- **AI Friendly** — Simple XML structure suited for LLM code generation
- **Declarative** — Describe slides in XML. No imperative API calls needed
- **Flexible Layout** — Flexbox-style layout powered by yoga-layout (VStack / HStack)
- **Rich Nodes** — 18 built-in node types: charts, flowcharts, tables, timelines, and more
- **Schema-validated** — Runtime validation with Zod schemas
- **PowerPoint Native** — Full use of native PowerPoint shape features
- **Pixel Units** — Intuitive pixel-based sizing (internally converted to inches at 96 DPI)
- **Master Slide** — Define headers, footers, and page numbers once and apply to all slides
- **Accurate Text Measurement** — Precise text measurement with opentype.js and bundled Noto Sans JP font

## Quick Start

Requires Node.js 18 or later.

```bash
npm install @hirokisakabe/pom
```

```typescript
import { buildPptx } from "@hirokisakabe/pom";

const xml = `
<VStack w="100%" h="max" padding="48" gap="24" alignItems="start">
  <Text fontSize="48" bold="true">Presentation Title</Text>
  <Text fontSize="24" color="666666">Subtitle</Text>
</VStack>
`;

const { pptx } = await buildPptx(xml, { w: 1280, h: 720 });
await pptx.writeFile({ fileName: "presentation.pptx" });
```

## Node List

| Node         | Description                                   |
| ------------ | --------------------------------------------- |
| Text         | Text display (font styles, decorations)       |
| Ul           | Bullet list                                   |
| Ol           | Numbered list                                 |
| Image        | Image (file path, URL, base64)                |
| Table        | Table                                         |
| Shape        | PowerPoint shape (178 types)                  |
| Chart        | Chart (bar, line, pie, area, doughnut, radar) |
| Timeline     | Timeline / roadmap                            |
| Matrix       | 2x2 matrix                                    |
| Tree         | Org chart, decision tree                      |
| Flow         | Flowchart                                     |
| ProcessArrow | Chevron-style process diagram                 |
| Pyramid      | Pyramid diagram                               |
| Line         | Line / arrow                                  |
| Layer        | Absolute positioning container                |
| VStack       | Vertical stack layout                         |
| HStack       | Horizontal stack layout                       |
| Icon         | Lucide icon                                   |

See the [Nodes](/nodes) for details. For color, font, and visual effect guidance, see the [Styling Guide](/styling-guide).

## Auto-Fit

When content exceeds the slide height, pom automatically adjusts it to fit (enabled by default). See [API Reference](./api-reference.md#autofit) for adjustment priority and configuration.

## For AI Agent Developers

If you are building an AI agent that generates PPTX with pom, include the content of [`llm.txt`](/llm.txt) in your system prompt. It provides a compact XML reference designed for LLMs.
